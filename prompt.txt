## üõ† TECHNICAL REQUIREMENTS

### Framework & Libraries
```
- React 18+ (or vanilla JS if preferred)
- Tailwind CSS for styling
- Framer Motion OR GSAP for animations
- Intersection Observer for scroll triggers
- Lenis or Locomotive Scroll for smooth scrolling (optional)
```

### Performance
- Lazy load images and heavy sections
- Optimize animations for 60fps
- Use CSS transforms over layout-triggering properties
- Implement proper will-change hints
- Core Web Vitals compliant (LCP < 2.5s, CLS < 0.1)

### SEO Requirements
- Semantic HTML5 structure
- Proper heading hierarchy (single H1, logical H2/H3)
- Meta tags (title, description, OG tags)
- Schema.org markup for Service/Offer
- Fast loading (performance is ranking factor)
- Mobile-first responsive design

### Accessibility
- Keyboard navigation support
- Reduced motion media query respect
- Sufficient color contrast
- Focus states on interactive elements
- Alt text for images
- ARIA labels where needed

## üé¨ ANIMATION SPECIFICATIONS

### Scroll Reveals
```javascript
// Intersection Observer config
threshold: 0.2
rootMargin: "0px 0px -100px 0px"

// Animation: fade up
from: { opacity: 0, y: 60 }
to: { opacity: 1, y: 0 }
duration: 0.8s
easing: cubic-bezier(0.22, 1, 0.36, 1)
stagger: 0.1s (for lists)
```

### Number Counters
```javascript
// Animate from 0 to target
duration: 1.5s
easing: easeOut
format: currency (‚Ç¨) or percentage (%)
trigger: when element is 50% visible
```

### Hover Effects
```javascript
// Cards
scale: 1.02
translateY: -8px
boxShadow: elevated
transition: 0.3s ease

// Buttons
scale: 1.05
glow: add subtle glow
transition: 0.2s ease
```

### Magnetic Cursor (CTAs)
```javascript
// Button follows cursor within radius
radius: 100px
strength: 0.3
transition: 0.5s ease-out
```
